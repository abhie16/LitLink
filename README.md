# LitLink - A Book Listing Website
## Website For Book Listing of College Library Books
### Features
- Book Listing
- Book Search
- Book Details
- Admin Panel
    - Admin Login
    - Add Book
    - Update Book
    - Delete Book
- User Panel
    - User Login
    - User Registration
    - User Profile
    - User Logout

### Technology Used
- nodejs
- expressjs
- mongodb
- mongoose
- reactjs
- redux
